# Simple Library Management System using JavaFX

## Project Description
This project is a simple Library Management System developed using Java and JavaFX.  
The system provides a graphical interface where screens are displayed using JavaFX, while the main operations such as adding, deleting, and borrowing books are handled through the console.

## Users
The system has two types of users:
1. Librarian  
2. Library Member  

## Login System
The system starts with a login screen where users enter:
- Username  
- Password  

Based on the user type, the system opens the appropriate dashboard.

## Librarian Permissions
The librarian can:
- Add books (Console)
- Delete books (Console)
- Search for books
- Update book information (Console)
- Logout

## Member Permissions
Library members can:
- Search for books
- View books
- Logout

## System Screens
The system contains three main screens:
- Login Screen  
- Librarian Dashboard  
- Member Dashboard  

Note: The screens are displayed using JavaFX, while most operations such as add, delete, and update are performed through the console.
